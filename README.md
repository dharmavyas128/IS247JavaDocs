From Intellij = Tools => Generate javaDocs => select Whole project => ok

JavaDoc is a documentation generator that is included with certain development applications and resources. 
It can be used with Intellij to create a page that includes documentation of a project that is cloned on 
github. 

A key tool that JavaDoc is able to use is to extract comments from a developers code and add it to the documentation 
page in a way that enables other developers or outsiders to understand the project or code. This inlcudes different methods
and functions that the project may have.

Other tools include a search box to search for something specific in the project, automatic summary pages, and tag support
for things such as parameters and returns that the JavaDoc can recogonise with implementation.
